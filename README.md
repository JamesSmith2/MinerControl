# MinerControl
MinerControl software for MinerMonitoring

# Requirements

- This tool required .NET Framework 4.8 installed (https://dotnet.microsoft.com/download/dotnet-framework/thank-you/net48-web-installer)

# Supports

- Excavator
- SGMiner
- BMiner
- PhoenixMiner
- Claymore Dual miner
- ZM Dstm
- EWBF
- EthMiner
- Cast-XMR
- XMR-Stak
- CCMiner (Forks also)
- Gateless gate
- NoncerPro
- TRex
- GMiner
- XMRig
- TeamRedMiner
- TTMiner
